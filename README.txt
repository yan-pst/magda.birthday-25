HappyBirthdayMagda Premium 🌻

Cara upload ke GitHub Pages:
1. Extract ZIP ini.
2. Buat repository baru di GitHub, misalnya: happy-birthday-magda
3. Upload semua isi folder HappyBirthdayMagda_Premium:
   - index.html
   - style.css
   - script.js
   - assets/
   - README.txt
4. Buka Settings → Pages.
5. Source: Deploy from a branch.
6. Branch: main, folder: /root.
7. Tunggu 1–2 menit sampai link aktif.

Catatan:
- Video sudah dimasukkan di assets/video/birthday_message.mp4.
- Foto sudah ada 25 di assets/photos.
